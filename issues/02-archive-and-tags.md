# Issue 2: 归档页 + 标签聚合系统

## Parent

PRD: 个人博客网站 — 复刻 yangzhiping.com (`../PRD.md`)

## What to build

构建两个内容导航页面：归档页按时间线组织所有文章，标签系统按主题聚合文章。同时让文章页的标签可点击跳转到对应标签聚合页。

## Acceptance criteria

- [ ] `/archive` 页面按年份/月份倒序列出所有文章
- [ ] 每个时间分组显示该时段内的文章数量
- [ ] `/tags` 页面展示所有标签（标签云样式，标签大小反映文章数量）
- [ ] 点击标签进入 `/tags/<tag>` 页面，显示该标签下的所有文章列表
- [ ] 文章详情页的标签链接可点击，跳转到对应的标签聚合页
- [ ] 标签页面导航栏可访问
