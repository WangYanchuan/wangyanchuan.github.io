# Issue 4: 中文排版优化 + 响应式美化

## Parent

PRD: 个人博客网站 — 复刻 yangzhiping.com (`../PRD.md`)

## What to build

对博客进行视觉打磨，重点优化中文阅读体验和移动端适配。风格参考阳志平网站：大量留白、清晰的信息层级、专注内容的排版。

## Acceptance criteria

- [ ] 中文字体栈优化（优先系统原生中文字体，如 PingFang SC、Noto Sans SC）
- [ ] 正文字号、行高、字距适合中文长文阅读（建议 16-18px，行高 1.8-2）
- [ ] 中文标点排版正确处理（引号、破折号、省略号等）
- [ ] 文章页有合适的最大宽度（~720px），避免行过长
- [ ] 首屏有适当的留白和呼吸感
- [ ] 移动端（<768px）页面内容完整可读，无水平滚动
- [ ] 文章图片响应式缩放
- [ ] 配色方案简洁、对比度适中（建议浅色主题，深色文字，点缀色克制）
- [ ] 代码块样式美观（背景色、圆角、行号）

## Blocked by

- #1 Hugo 项目初始化 + 首页与文章详情

## Type

HITL — 需要设计审阅确认配色和排版效果
