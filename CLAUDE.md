## Project Overview

This is a blog project under the "零基础AI大模型研发训练" course ecosystem. The blog publishes articles about LLM training, AI development, and related topics.

## Development Workflow

This project follows a 5-stage workflow using the available skills:

1. **`to-prd`** — Convert conversation context into a PRD (GitHub Issue)
2. **`to-issues`** — Break PRD into independently-grabbable vertical-slice issues
3. **`tdd`** — Red-green-refactor development loop for each issue
4. **`diagnose`** — Structured bug diagnosis when issues arise
5. **`improve-codebase-architecture`** — Refactor and improve code quality
